1. sqld.zip 파일의 압축을 풀고cre_scott.sql 파일과 cre_kleague.sql 파일을 c:\db_test\sqld 폴더에 복사해 둔다.
2. run SQL CommandLine을 실행한 후 다음과 같이 cre_scott.sql을 실행하여 scott(암호는tiger) 사용자를 생성한다.
   @c:\db_test\sqld\cre_scott.sql
4. run SQL CommandLine을 종료한다.(exit)
5. SQL Developer를 실행한 후 왼쪽 상단 초록색 + 기호를 클릭해서 새 접속을 생성한다.
    접속이름 : K리그   사용자 : scott 암호 tiger
6. K리그로 접속한 상태에서 cre_kleague.sql 파일을 열고 파일의 내용을 SQL Developer 워크시트에 모두 복사하여 붙여넣기 한다.
7. 워크시트 상단의 왼쪽에서 두 번째 아이콘인 스크립트 실행 아이콘을 눌러 화면의 모든 명령문을 실행한다.
8. 다음 명령을 실행하여 테이블과 데이터가 정상적으로 생성되었는지 여부를 확인한다.
SELECT * FROM emp;
SELECT * FROM player;
SELECT * FROM schedule;
SELECT * FROM stadium;
SELECT * FROM team;
